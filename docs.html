Introduction

AutoGUI is a Python library that bridges JSON-based design with the power of Kivy/KivyMD. Define user interfaces declaratively using JSON files, then interact with them using a simple, pythonic API.

Runtime-aligned features:

  • Responsive scaling based on window size
  • Material-styled components via KivyMD
  • Minimum window size locking
  • Absolute positioning with FloatLayout

Setup & Initialization

Create a Python script (e.g., main.py) and a JSON layout file.

Python code

from NVLib.Components.GUI.AutoGUI import AutoGUI

# 1. Initialize the App
app = AutoGUI()

# 2. Configure (Optional)
app.set_title("My Application")
app.set_background("#ffffff")

# 3. Load Layout
app.build_gui("layout.json")

# 4. Define Logic
def on_submit():
    print(f"User typed: {app.my_input.text()}")

# 5. Bind Events
app.my_button.on_click(on_submit)

# 6. Run
app.run()

Canvas configuration

The root object in your JSON file. Defines the initial window properties.

{
  "canvas": {
    "width": 800,
    "height": 600,
    "title": "My App Title"
  },
  "components": [ ... ]
}

Component reference

Each component lists supported JSON properties JSON and pythonic methods Python. This page is validated to match the current runtime behavior.

Button

A standard clickable button with ripple effects.

JSON properties

PropertyTypeDescription
typeString"Button"
textStringLabel on the button.
backgroundColorHexButton background color.
textColorHexButton text color.
fontSizeIntFont size (default 14).
cornerRadiusIntRounded corners for button container.
boldBoolEmphasize label (mapped to a larger style).

Python API

MethodDescription
.on_click(func)Bind a function to the click event.
.text(new_text=None)Get or set the button label.
.set_visible(bool)Show or hide the button.
.bg_color(hex)Set background color.
.text_color(hex)Set label color.
.font_size(int)Set font size.
.bold(bool)Apply emphasis.

Label

Text or icon display. Supports Material Symbols if iconName is set.

JSON properties

PropertyTypeDescription
typeString"Label"
textStringText content (ignored if iconName is set).
iconNameStringMaterial icon name (optional).
textColorHexText/icon color.
backgroundColorHexBackground color.
fontSizeIntFont size.
boldBoolBold text (for text labels).

Python API

MethodDescription
.text(new_text=None)Get or set label text.
.set_visible(bool)Show or hide the label.
.bg_color(hex)Set background color.
.text_color(hex)Set text/icon color.
.font_size(int)Set font size.
.bold(bool)Apply bold to text labels.

Panel / CardView

Container elements used for grouping other widgets and creating backgrounds.

JSON properties

PropertyTypeDescription
typeString"Panel" or "CardView"
backgroundColorHexBackground fill color.
cornerRadiusIntRoundness of corners.
elevationInt(CardView only) Card elevation.
textString(CardView optional) Centered label text.
textColorHex(CardView optional) Label color.
fontSizeInt(CardView optional) Label font size.
boldBool(CardView optional) Bold label.
borderColorHex(CardView only) Border color.
borderWidthInt(CardView only) Border width.
opacityFloatContainer opacity.
visibleBoolInitial visibility.

Note: Place Panel/CardView before other components in the JSON list to ensure they render behind content. Components inside a container are detected by absolute bounds.

Python API

MethodDescription
.set_visible(bool)Show or hide.
.bg_color(hex)Set background color.
.opacity(float)Set opacity.
.border(width, hex, visible=True)(CardView) Configure border.

Image

Base64 image rendering inside a Card container.

JSON properties

PropertyTypeDescription
typeString"Image"
srcStringBase64 data URL (e.g., data:image/png;base64,...).
fitString"fill" | "contain" | "cover" | "none".
cornerRadiusIntCard corner radius.
backgroundColorHexCard background color.
opacityFloatCard opacity.
visibleBoolInitial visibility.

Python API

MethodDescription
.set_visible(bool)Show or hide.
.bg_color(hex)Set card background color.
.opacity(float)Set card opacity.

Note: If src is invalid, a simple “Image Error” label is shown.
